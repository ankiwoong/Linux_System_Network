* date

날짜와 시간을 출력

```shell
[root@linux1 ~]# date
2020. 09. 01. (화) 20:08:09 KST
```

* clear

화면을 지운다.

```shell
[root@linux1 ~]# clear
[root@linux1 ~]# 
```

* man

명령 사용법을 화면에 출력한다.

```shell
[root@linux1 ~]# man clear
[root@linux1 ~]# 
```

* passwd 명령

사용자 계정의 비밀번호를 변경한다.

```shell
[root@linux1 ~]# passwd
root 사용자의 비밀 번호 변경 중
새  암호:
새  암호 재입력:
passwd: 모든 인증 토큰이 성공적으로 업데이트 되었습니다.
```
